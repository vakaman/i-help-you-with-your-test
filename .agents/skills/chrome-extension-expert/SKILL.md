# Chrome Extension Expert Skill

This skill provides expertise in designing, building, and optimizing Google Chrome extensions using modern manifest V3 standards.

### Expert knowledge in:
- `manifest.json` (V3 configuration, host permissions, web accessible resources).
- Background scripts (Service Workers) and lifestyle management.
- Content scripts (injecting JS/CSS and accessing page DOM).
- Popup and options pages (interactive UI and data persistence).
- Messaging API (communication between popup, background, and content scripts).
- Storage APIs (`chrome.storage.local`, `sync`).
- Integration with external APIs and services.
- Building custom UI with modern tools (Preact, React, or vanilla JS/CSS).
- Security best practices for extensions.

### Tools to use:
- `chrome.devtools` for debugging.
- Webpack/Vite for bundling (if complex).
- TailwindCSS/PostCSS for styling.

### Examples:
- "How do I inject a content script into a specific URL?"
- "Set up a shared state between background and popup."
- "Optimize the memory usage of my service worker."
